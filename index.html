Conditionals Presentation & Practice

If this happens...
then that happens.

A bright, memorable way for you to revise condition + result, then move into zero, first, second, third and a light introduction to mixed conditionals.

0 = facts 1 = real future 2 = imaginary now/future 3 = unreal past Mixed = time bridge

Condition

The main clause. It is the consequence.

... I will do well tomorrow.
1 / Core concept
Condition + Result

Condition and result

🔵 Condition = trigger

This is the if-clause.

If you heat ice, ...
If it rains tomorrow, ...
→

🟢 Result = consequence

This is the main clause.

... it melts.
... we will stay inside.

Golden rule 1

If can come first or second.

If I see her, I'll tell her.
I'll tell her if I see her.

Golden rule 2

Use a comma only when the if-clause comes first.

If I see her, I'll tell her.

Golden rule 3

Always ask: Is this real? imagined? past? future? always true?

3 / Forms
Zero + First

Zero + first

ZERO CONDITIONAL

For facts, rules, routines, and things that are generally true

If + present simple, present simple
If people don't drink water, they get thirsty.
always true science instructions
FIRST CONDITIONAL

For real future possibilities

If + present simple, will + base verb
If it rains tomorrow, we'll cancel the match.
possible future plans warnings

Fast contrast

Zero: If you mix blue and yellow, you get green.

First: If you mix these two paints, you'll get green.

Meaning note

Contrast general truth with specific future possibility. That difference solves many mistakes.

4 / Forms
Second + Third

Second + third

SECOND CONDITIONAL

Imaginary present or future

If + past simple, would + base verb
If I had a million euros, I would travel for a year.
imaginary now dreams advice
THIRD CONDITIONAL

Imaginary past / regret / criticism

If + past perfect, would have + past participle
If she had studied more, she would have passed.
past regret can't change now looking back

Important meaning clue

Second = not real now.

Third = not real in the past.

Useful self-check

Ask: Can we still change it?

If yes or maybe yes → second. If no, because it is finished → third.

5 / Stretch
Mixed conditionals

Mixed conditionals = a bridge between two times

PAST → PRESENT

A past action affects the present

If + past perfect, would + base verb
If I had gone to bed earlier, I wouldn't be so tired now.
PRESENT → PAST

A present situation or state affected the past

If + past simple, would have + past participle
If I were more organised, I would have finished the project yesterday.

7 / Practice
Scenario set 1

Scenario practice 1

Choose the best conditional, use the prompt verbs, say your sentence, then flip the card to check.

1. Present personality, past result

Noah is so disorganised that yesterday he forgot his notes at home.

Use: be / not forget
Best fit: Mixed conditional.
If Noah were more organised, he wouldn't have forgotten his notes yesterday.
2. Tomorrow's forecast

The forecast says there may be snow tomorrow during football practice.

Use: snow / cancel
Best fit: First conditional.
If it snows tomorrow, we'll cancel football practice.
3. Lost essay

Lara didn't save her file, so she lost her essay.

Use: save / not lose
Best fit: Third conditional.
If Lara had saved her file, she wouldn't have lost her essay.
4. Too much headphone time

Students use headphones for too long. Their ears hurt afterwards.

Use: use / hurt
Best fit: Zero conditional.
If students use headphones for too long, their ears hurt.
5. Island life

You imagine living on a Greek island all year round.

Use: live / swim
Best fit: Second conditional.
If I lived on a Greek island, I would swim every day.
6. Late night, tired now

I went to bed at 2 a.m., so now I'm exhausted.

Use: not go / not be
Best fit: Mixed conditional.
If I hadn't gone to bed so late, I wouldn't be exhausted now.
